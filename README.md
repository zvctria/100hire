# 100hire
Portfolio Project - 100Hires

This project was created as part of the application process for the Junior Growth Marketing Specialist position at 100Hires.1. Tools Installed

Here is the list of tools I installed and configured for this project:
Cursor IDE: Used as the primary code editor.
Claude Code (Extension): Anthropic's agentic coding assistant. I used this for its robust "plan-then-execute" workflow, which ensures that complex code changes are proposed, reviewed, and approved before being applied to the codebase.
Codex (Extension): OpenAI's autonomous coding agent. I integrated this to handle "fire-and-forget" tasks—offloading repetitive implementation work and debugging to an autonomous agent, which significantly sped up my development loop.
GitHub: Used as a repository to store the project code and documentation.

Here are the steps I followed to set up this development environment:
Installed Cursor IDE via the official Cursor website.
Searched for and installed the "Claude Code" add-on via the Extensions tab in Cursor, then logged in.
Searched for and installed the "Codex" add-on via the Extensions tab in Cursor, then logged in.
Created a new public repository on GitHub.
Connected my local repository on my computer to the GitHub repository.
